# uShip
First Commit of Android Code

Development Tools and SDK: 
Android Studio 2.3 version
SDK: 25 API or Android 6 Marshmallow 

Android app for uShip to display list of 151 original pokemons from a REST based API
On click of specific pokemon from list view, a itemOnclick listener launches another activity
to display the front sprite image of the selected pokemon along with other features of name,
height, weight and its types

Mechanism to handle hardware back and title bar software back is also handled 
Check for network connection is handled in case of no internet, the app gracefully closes
